# Таблица SPA `React`

## Установка <a name="Installation"></a>

> Клонировать репозиторий
>    ```sh
>    git clone https://github.com/Hydrobee3000/Test-table.git
>    ```
>    
> Установить зависимости 
>    ```sh
>    npm install
>    ```
>    
> Запустить проект
>    ```sh
>    npm start
>    ```
